# Project Brief — LembonganFunTrip Website
**Domain:** funtriplembongan.com  
**Referensi Desain:** suntriplembongan.com  
**Target Live:** Awal Juni (musim ramai)  
**Estimasi Pengerjaan:** 2 minggu

---

## Gambaran Proyek

Website etalase digital untuk bisnis tour & sewa kendaraan di Nusa Lembongan. Target pengguna adalah wisatawan mancanegara (mayoritas Australia). Semua transaksi dilakukan via WhatsApp — website berfungsi sebagai media discovery, membangun kepercayaan, dan mengarahkan tamu ke WA.

---

## Target Pengguna

- Wisatawan asing, mayoritas dari Australia
- Tidak perlu login / buat akun
- Langsung diarahkan ke WhatsApp untuk booking

---

## Bahasa

- **English only** (konten seluruhnya bahasa Inggris)
- Teks deskripsi package sebaiknya ditulis oleh client

---

## Produk / Layanan yang Ditawarkan

| Layanan | Tipe | Catatan |
|---|---|---|
| Buggy Car | Sewa | Hanya di Nusa Lembongan, tidak bisa ke Nusa Ceningan |
| Motor | Sewa | Bisa sampai Nusa Ceningan via jembatan |
| Snorkeling | Aktivitas | - |
| Island Tour | Tour | - |

> **Penting:** Tidak diperlukan SIM di Nusa Lembongan — ini nilai jual yang perlu ditonjolkan.

---

## Alur Pengguna (User Flow)

```
Tamu buka website
    ↓
Homepage (hero + overview packages + reviews)
    ↓
Pilih: Tour & Activities / Blog & Stories / About
    ↓
Halaman Detail Package
(foto, harga, deskripsi, rules, CTA)
    ↓
Tap "Book via WhatsApp"
(pre-filled message: nama, tanggal, package)
    ↓
Konfirmasi & transaksi di WhatsApp
Data pemesanan disimpan di WA (tidak perlu sistem tambahan)
```

---

## Fitur Website

### Halaman Publik (Tamu)
- [ ] Homepage — hero section, overview package, review singkat
- [ ] Halaman Tour & Activities — list semua package dengan foto & harga
- [ ] Halaman Detail Package — deskripsi lengkap, foto, rules, tombol WA
- [ ] Blog / Stories — artikel dan real guest reviews
- [ ] Halaman About

### Booking
- [ ] Tombol "Book via WhatsApp" di setiap package
- [ ] Pre-filled WA message (nama package, tanggal, dll)
- [ ] **Tidak ada** sistem notifikasi di website setelah transaksi selesai
- [ ] **Tidak ada** login untuk tamu

### Admin Panel
- [ ] Login admin (owner)
- [ ] Tambah / edit / hapus package
- [ ] Preview konten sebelum publish
- [ ] Tambah / edit artikel dan review
